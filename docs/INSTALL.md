# 🏗️ INSTALL

How to download, test and run.

## Download

```bash
git clone https://github.com/helvm/helms.git
cd helms
```

## Cabal

Compile and run with `cabal`:
```bash
cabal clean && cabal build && cabal test
cabal run helms
```
